# Chronical 5 - AI-Powered Business & Creativity Assistant

**By Infinite Chronicles**

Chronical 5 is a production-ready SaaS platform helping people transform ideas into successful projects, businesses, brands, products, and stories through AI-powered guidance and structured roadmaps.

## 🚀 Core Features

- **Business Builder**: Generate complete startup roadmaps from ideas
- **Story Creator**: Create characters, worlds, chapters, and scripts
- **Idea Generator**: Generate startup, app, product, and content ideas
- **AI Chat Assistant**: Modern conversational interface with multiple AI models
- **Image Generation**: Create logos, branding, product concepts, and artwork
- **WhatsApp Integration**: Chat with Chronical 5 from WhatsApp

## 📊 Tech Stack

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js
- **Payments**: Razorpay
- **Storage**: Cloudinary
- **AI Models**: Gemini, Gemma, Qwen
- **Deployment**: Vercel

## 🛠️ Quick Start

```bash
git clone https://github.com/Chronical-5/chronical-5-platform.git
cd chronical-5-platform
npm install
cp .env.example .env.local
npm run prisma:generate
npm run prisma:migrate
npm run dev
```

Visit http://localhost:3000
